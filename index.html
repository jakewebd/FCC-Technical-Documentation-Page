CSS Documentation

Introduction

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML (including XML dialects such as SVG, MathML or XHTML). CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

CSS is among the core languages of the open web and is standardized across Web browsers according to W3C specifications. Previously, development of various parts of CSS specification was done synchronously, which allowed versioning of the latest recommendations. You might have heard about CSS1, CSS2.1, CSS3. However, CSS4 has never become an official version.

From CSS3, the scope of the specification increased significantly and the progress on different CSS modules started to differ so much, that it became more effective to develop and release recommendations separately per module. Instead of versioning the CSS specification, W3C now periodically takes a snapshot of the latest stable state of the CSS specification

  • Understanding basic color theory and contrast ratios

Getting started

To get started with CSS open up a notepad document or a source code editor of your choice.

You want to start by creating a <style> element in your HTML document.

This will be where you enter the CSS selectors and properties to style the content.


   <style>

   selector {

   CSS-property: ;

  }

   </style>

Another way to style a page is to link the HTML document to an external CSS file.

This is done by using the link element in the HTML head.


  <link rel="stylesheet" type="text/css" href="file.css" />

By linking to an external CSS file you dont have to put everything in between <style> elements.

CSS Syntax

A CSS rule consists of a selector and a declaration block.

A selector points to the target element you want to style.

A declaration block contains one or more declarations seperated by semicolons. ;

Each declaration contains a CSS property name and a value, seperated by a colon. :

Multiple declaratons are seperated by semicolons, and declaration blocks are surrounded by curly braces. {}


  selector {

  CSS-property-name: value;

  }

Selectors

CSS Selectors are used to target an element so you can style it.

A few basic CSS selectors are:

  • Class selectors
  • ID selectors
  • Element type selectors
  • Universal Selector

The basic selectors shown above are used as follows:


  .class {CSS-property: ;}

  #ID {CSS-property: ;}

  h1 {CSS-property: ;}

  * {CSS-property: ;}

CSS Comments

CSS Comments are not displayed in the browser, but they can help you document your source code.

Comments are used to explain the code, and may help you if you need to edit the source code at a later date.

CSS Comments can be placed on the HTML page in between <style> elements, or on an external CSS file.

The syntax is as follows:


  /* this is a single-line comment */

  /* this is a
  multi-line comment */

Backgrounds

CSS backgrounds are used to add background effects for elements.

A few CSS background properties are:

  • background-color
  • background-image
  • background-repeat
  • background-position

The background-color property is used to set the color of the elements background. The background-image property is used to set an image as the background for an element. The background-position property is used to set where in the element the background should start from. The background-repeat property states that if the background is undersized it wont repeat multiple times across the screen/element.

The syntax to use these properties is as follows:


  selector {

  background-color: color;

  background-image: url(linktoimage);

  background-repeat: no-repeat;

  background-position: top left;

  }

Height/Width

The CSS height and width properties are used to set the height and width of an element.

The height and width properties do not include padding, borders, or margins. It sets the height/width of the area inside the padding, border, and margin of the element.

The height and width properties may have the following values:

  • auto - this is the default value. the browser calculates the width/height
  • length - defines the height/width in px, cm, etc.
  • % - defines the height/width in a percentage of the containing element

The syntax for setting the height/width is as follows:


  selector {

  height: value;

  width: value;

  }

Margins

The CSS margin properties are used to create space around elements, outside of any defined borders.

CSS has properties for specifying the margin for each side of an element.

Those properties are:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

All margin properties can have the following values:

  • auto - this is the default value. the browser calculates the width/height
  • length - defines the height/width in px, cm, etc.
  • % - defines the height/width in a percentage of the containing element

Setting a margin on each side of an element is done as follows:


  selector {

  margin-top: value;

  margin-right: value;

  margin-bottom: value;

  margin-left: value;

  }

You can use the margin - shorthand property to shorten the code and specify all margin side properties in one property.

The way it works is that if the margin property has four values it will create a margin in the order of top, right, bottom, left.

If it has 3 values it will create it with the first value being the top margin, the second value being the margin for right and left, and the third value being the bottom margin.

If it has 2 values it will create a margin using the first value for the top and bottom margins, and the second value for the right and left margins.

If it has one value it will set the same size margin for all sides.

the syntax for using the margin - shorthand property is as follows:


  selector {

  margin: value value value value;

  }
